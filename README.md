# PRODIGY_DS_01

Task-01: Population Distribution Visualization

In this task, a histogram was created to visualize the distribution of population across countries for the year 2020 using World Bank data.

Tools Used-
- Python
- Pandas
- Matplotlib
- Google Colab

Output-
The histogram shows a right-skewed distribution, indicating that most countries have smaller populations, while a few countries have very large populations.

DATASET-
https://data.worldbank.org/indicator/SP.POP.TOTL
